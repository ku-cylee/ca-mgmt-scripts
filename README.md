# CA Management Server Setup

## Environment Setup

### BSC Compiler Setup
* Check and update `SHARED_DIR` and `TARBALL_URL` variables in `envs/bsc`
    - BSC github repo: https://github.com/B-Lang-org/bsc
    - Check *latest release* and *system version* and update accordingly.
* Execute: `sudo ./envs/setup-bsc`
* Login again and check if `bsc` command works and the assignment projects are compiled properly.
